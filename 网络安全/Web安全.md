# Web安全



## 1. 使用 kali 工具渗透

Kali Linux是一个开源的、基于Debian的Linux发行版，面向各种信息安全任务，如渗透测试、安全研究、计算机取证和逆向工程。

官网地址：https://www.kali.org/



### 1.1 docker安装kali

```shell
docker pull kalilinux/kali-rolling

docker run --name kali -it kalilinux/kali-rolling
```
